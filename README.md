# Jeux

Petits jeux web pédagogiques (esprit critique, fake news, biais cognitifs, culture générale…).
Chaque jeu est autonome : il suffit d'ouvrir son fichier `index.html` (ou le `.html` du jeu) dans un navigateur.

## Jeux inclus

| Dossier / fichier | Thème |
|---|---|
| `radarnaque/` | Radar'naque — repérer les arnaques (SMS, e-mails, appels, deepfakes…) |
| `radar-desinfo/` | Repérer la désinformation |
| `chasse-aux-biais/` | Chasse aux biais cognitifs |
| `arene-rhetorique/` | Arène de rhétorique |
| `grand-oral/` | Entraînement au grand oral |
| `repare-la-une/` | Réparer la une d'un journal |
| `Ilusions d'optique/` | Illusions d'optique |
| `mes des mal information/` | Dés de la mal-information |
| `Arcade quizz/` | Quiz arcade fake news |
| `echecs.html` | Ouvertures aux échecs |
| `mine.html` | Mine la Planète (extraction de ressources) |
| `terre_ronde_plate.html` | La Terre ronde ou plate ? |
| `biais_confirmation_cm1.html` | Biais de confirmation (CM1) |
| `Planète connaissance.html` | Quiz culture générale |
| `notes_google.html` | Avis « style Maps » (démo de faux avis) |
| `snake-fakenews.html` | Snake — Fake News & esprit critique |
